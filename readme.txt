The Grade Matcher script is a simple tool designed to assign grades to students or athletes based on their accumulated points. This script is helpful for educators, coaches, or anyone involved in assessing performance and providing corresponding grades.

Getting Started
1. Clone the repository to your local machine.
2. Ensure you have Python installed.
3. Run the script in your preferred Python environment.
4. Input students and their points in the same order in name and points sections (You can adjust point range as per your requirements).
5. View the generated grades.
